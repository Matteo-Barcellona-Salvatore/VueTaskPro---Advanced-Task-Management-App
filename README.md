#  VueTaskPro

Modern task management application built with Vue 3, TypeScript, and Tailwind CSS.

##  Features

-  Real-time dashboard with statistics
-  Dark/Light mode toggle
-  Advanced search and filtering
-  Interactive charts
-  Local storage persistence
-  Fully responsive design

##  Tech Stack

Vue 3 • TypeScript • Pinia • Vue Router • Tailwind CSS • Chart.js • Vite

##  Installation

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/vue-task-pro.git
cd vue-task-pro

# Install dependencies
npm install

# Run development server
npm run dev
```

##  Build

```bash
npm run build
npm run preview
```

##  Preview

Visit `http://localhost:5173` to see the app in action!
